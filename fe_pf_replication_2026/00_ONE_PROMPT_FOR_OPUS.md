# FE Protein Folding Scaling Study — Run Instructions

## What This Is
Option B full scaling sweep: Forgetting Engine vs Monte Carlo on 3D HP lattice protein folding.
7 lengths (20–50), 2000 trials per algorithm per length, identical parameters throughout.

## Prerequisites
- Python 3.8+
- `pip install numpy scipy pandas matplotlib seaborn tqdm`

## Run the Full Experiment

### Step 1: Phase A (Pilot) — ~45 minutes
```bash
cd fe_pf_replication_2026
python run_experiment.py --phase A
```
This runs 200 MC + 200 FE trials per length and locks the success thresholds E*(L).
Thresholds are saved to `results/pilot_thresholds.json`. Do NOT edit this file.

### Step 2: Phase B (Main) — ~3-4 hours
```bash
python run_experiment.py --phase B
```
This runs 2000 MC + 2000 FE trials per length using the locked thresholds.
Results are saved per-length as `results/phase_B_L{length}.csv`.

### Step 3: Analyze
```bash
python analyze_results.py
```
Produces:
- `manuscript_outputs/scaling_table.csv` — Publication table
- `manuscript_outputs/scaling_plot.png` — Three-panel figure (absolute rates, Δ(L), R(L))
- `manuscript_outputs/stats_summary.json` — Full statistical summary

### Or: Run Everything at Once (~4 hours)
```bash
python run_experiment.py --all
python analyze_results.py
```

## Run a Single Length (for testing)
```bash
python run_experiment.py --phase A --length 20
python run_experiment.py --phase B --length 20
python analyze_results.py
```

## What the Experiment Tests

**H₀**: R(L) is constant across L.
**H₁**: R(L) increases with L.

Primary metric: Δ(L) = FE_success% − MC_success%
Trend test: Fit Δ(L) = a + bL, test b > 0 with permutation test.

If b > 0 with statistical significance → "Complexity Amplification Effect" is real.

## Rules (Do Not Violate)
1. Do NOT change config.json parameters between lengths
2. Do NOT edit pilot_thresholds.json after Phase A
3. Do NOT reuse seeds across lengths
4. Report absolute success rates alongside ratios
